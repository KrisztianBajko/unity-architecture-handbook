# Unity Architecture Handbook

An interactive handbook teaching professional Unity architecture, software design principles, and maintainable game development.

The handbook is designed to help Unity developers move beyond writing features and begin designing scalable, modular architectures.

---

## Live Website

https://krisztianbajko.github.io/unity-architecture-handbook/

---

## Available Chapters

### Chapter 1 — Why Most Unity Projects Become Spaghetti

Learn why projects become difficult to maintain and how architectural debt grows over time.

Topics include:

- Complexity Growth
- Hidden Dependencies
- Manager Explosion
- Coupling
- Architectural Debt

---

### Chapter 2 — Modularity & Service Architecture

Learn how professional Unity projects organize systems into modules with clear responsibilities.

Topics include:

- Bootstrappers
- Services
- Events
- Modules
- Explicit Dependencies
